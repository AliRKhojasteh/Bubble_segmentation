# Contributing to Bubble Segmentation

We welcome your contributions to improve bubble segmentation in fluid experiments!

## Additional Contributions

We are particularly interested in:

- **Test cases for bubble detection**: different bubble configurations, size distributions, void fractions, and lighting conditions from various experimental setups (channels, pipes, reactors, rising bubbles).
- **Experimental snapshots**: images from bubbly flow experiments that can help improve and validate the fine-tuned model.
- **Pipeline improvements**: enhancements to the automatic mask generation, post-processing, or interactive editing workflows.

For contributing test cases or data, please contact the maintainers or submit a pull request with a detailed description of your additions.

## How to Contribute

We use GitHub to host code, track issues, and accept pull requests. All code changes happen through pull requests, and we actively welcome your contributions:

1. Fork the repository and create your branch from `main`.
2. Add tests if you've added code that should be tested.
3. If you've changed APIs, update the documentation.
4. Ensure the test suite passes.
5. Make sure your code lints.
6. Issue that pull request!

## Report Bugs

Report bugs using GitHub [Issues](https://github.com/AliRKhojasteh/Bubble_segmentation/issues). When reporting a bug, please include:

- A quick summary and/or background
- Steps to reproduce (be specific!)
- What you expected would happen
- What actually happens
- Notes (e.g., why you think this might be happening, things you tried that didn't work)
